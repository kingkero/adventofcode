# Personal solutions to [Advent of Code](https://adventofcode.com)

## 2023

### Go

**Run all days:**

```bash
go run main.go
```

**Run all tests:**

```bash
go test ./...
```

**Run specific day:**

```bash
go run main.go --day 14
```

**Run specific day with demo data:**

```bash
go run main.go --day 14 --demo
```
